TwitchBot for every language

Our goal is to create a community of Twitch users who don't have to worry about the language barrier. We want to translate TwitchBot's UI into multiple other languages, and we need the community's help to do so.

How it works

We'll send you a collection of JSON files that have the existing sentences in English. All you have to do is translate the english parts to the languge that you've applied for.

{
 "help": {
  "partners": {
   "title": "TwitchBot Partners",
   "value": "We're looking for passionate Twitch streamers who use TwitchBot. Apply and get exclusive rewards: https://twitch.disgd.pw/partners"
  }
 }
}

For example, if you're translating to Japanese, you'll translate the file to something like this:

{
 "help": {
  "partners": {
   "title": "TwitchBotパートナー",
   "value": "TwitchBotを使用している情熱的なTwitchストリーマーを探しています。 https://twitch.disgd.pw/partnersを適用し、排他的な報酬を得る"
  }
 }
}
