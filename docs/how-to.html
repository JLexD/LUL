How do I use TwitchBot?

A comprehensive guide on using TwitchBot in your Discord server.

What is TwitchBot's prefix?

TwitchBot doesn't have a prefix like ! or ., all of TwitchBot's commands start with twitch.

How do I view a list of TwitchBot's commands?

You can see all the commands by typing twitch commands.

How do I upvote TwitchBot?

Go to this page, log in with Discord if you haven't already, and press Vote for this bot.

I just upvoted TwitchBot, why can't I listen to streams?

Make sure that you have voted in the last 24 hours, and that 5 minutes have passed since you upvoted last.

How do I add a notification when someone goes live?

You need to specify the streamer's Twitch name and the Discord channel you want it in. For example, typing twitch notif add #general ninja will add a notification for twitch.tv/ninja in the #general channel.

How do I add a custom message to a streamer notification?

Simply add the message after the command you would normally enter. For example: twitch notif add #streams xqcow Hey @everyone, XqcOW is live! https://twitch.tv/xqcow.

How do I set up live checking?

Type twitch live_check <role_name> with your role name and without the < and >. Make sure TwitchBot has the Manage Roles permission and that the role named TwitchBot is above the role you want to configure.
